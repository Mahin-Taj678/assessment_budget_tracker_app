# BudgetFlow - Next.js Personal Finance Tracker

A modern, full-stack personal finance and budget tracker designed with a stunning UI, dynamic animations, and real-time cloud syncing.

## ✨ Features
- **Dashboard Overview**: Track your total savings, monthly goals, and income vs. expenses at a glance.
- **Transaction Flow**: Easily log income and expenses, seamlessly categorized for your insights.
- **Budget Insights**: Interactive charts showing category breakdowns of your spending.
- **Save the Coins Mini-Game**: A gamified experience to save virtual coins within your budget goals!
- **Dark Mode Support**: Beautifully styled for both light and dark aesthetics.

---

## 🚀 Getting Started

Follow these instructions to clone the repository and run the application locally.

### 1. Clone the Repository
```bash
git clone https://github.com/Mahin-Taj678/assessment_budget_tracker_app.git
cd assessment_budget_tracker_app
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash
npm run dev
```

### 4. Open the Application
Navigate to [http://localhost:3000](http://localhost:3000) in your web browser. 

*Note: For detailed information about the application's architecture, Firebase database structure, and deployment instructions, please see the [DEPLOYMENT_ARCHITECTURE.md](./DEPLOYMENT_ARCHITECTURE.md) file.*
